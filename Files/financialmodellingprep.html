financialmodelingprep.com API¶

According to their website, Financial Modelling Prep is a new concept that informs you about stock markets information (news, currencies and stock prices). They always strive to give you the best and most updated information. They also give you free financial modelling methodology through their academy. Most investment banking firms follow their guidelines to get discounted cash flow statement of companies to see if they are undervalued, overvalued or simply at par value. You can find all financial models and valuation techniques that is used in corporate finance to get companies intrinsic valuation. Most private equity firm use financial modelling for decision making when it comes to hold, buy or sell a particular stock.

while it is a website worth exploring, we will be focusing on its API. Again, according to their website their API has the properties listed below:

  • We are the most accurate financial data API out there.
  • To make it simple we are a free stock API | historical data API | financial statements API
  • We update our financial statements every 10 Real-times every statement is audited, standardized, and up to date.
  • We cover NYSE, NASAQ, AMEX, EURONEX, TSX, INDEXES, ETFs, MUTUAL FUNDS, FOREX and CRYPTO.
  • We have real time stock price, we cover the fundamental data part of the stocks via providing income statement, balance sheet statement and cashflow statement quarterly and annually.
  • We have 1 minute, 15 minutes, 30 minutes, 1 hour and daily historical stock prices.

    To this I would add:

  • It's free (for now);
  • It's really easy to use;
  • It's well formatted and
  • It's well documented.

As in the case of web scrapping yahoo finance, I've writen all the code as functions This will make it easier for others to read, more consistent, decrease the likelihood of making mistakes and enable you to quickly build a module or a package that you can then use in other projects.

The function that makes the request to the website is get_jsonparsed_data. All the other functions call this one passing the specific url they want data from. This notebook contains 14 functions including the mentioned one. Most of them are almost identical to one another only differing in the data they make a request for. all the functions in this notebook have a docstring that explains what their inputs and outputs are. Immediately after the function code you'll find a line calling the function so that you can see what the output looks like.

def get_jsonparsed_data(url):
    """
    Receive the content of ``url``, parse it as JSON and return the object.

    Parameters
    ----------
    url : str

    Returns
    -------
    dict
    """
    response = urlopen(url)
    data = response.read().decode("utf-8")
    return json.loads(data)

    """
    Save a company's financial data from financialmodelingprep.com as 'csv' on the directory of choice.

    Saves all the finacial data available into three csv files which containing the information
    obtained by calling the functions listed below:\n
    `ticker`_profile.csv:`\n
        get_profile\n

            get_income_statement
            get_balance_sheet
            get_cash_flow
            get_ratios
            get_enterprise_value
            get_key_metrics
            get_financial_statement_growth
            get_DCF

            get_daily_price

    ----------
    `ticker` : `str`,
        The company's ticker symbol

    Returns
    -------
    pandas DataFrame
    """
    if not os.path.exists(directory):
        os.makedirs(directory)

    for ticker in tickers:
        if len(tickers) % tickers.index(ticker) == 0:
            print(str(len(tickers) % tickers.index(ticker) * 100) + '%')
        if not os.path.exists('/{}/{}_daily_price.csv'.format(directory, ticker)):

                df = get_profile(ticker)
                df.to_csv('/{}/{}_profile.csv'.format(directory, ticker))
                df = get_all_financials(ticker)
                df.to_csv('/{}/{}_financials.csv'.format(directory, ticker))
                df = get_daily_price(ticker)
                df.to_csv('/{}/{}_daily_price.csv'.format(directory, ticker))

                print("Something's wrong with ticketer:" + ticker)
